Esse projeto foi desenvolvido com as seguintes tecnologias e dependências:

- Angular
- Ionic
- SaSS
- react-google-login
- gapi-script

O sistema de estágio do IFSP, tem como objetivo melhorar os processos atuais que é realizado para gerenciar estágio.
